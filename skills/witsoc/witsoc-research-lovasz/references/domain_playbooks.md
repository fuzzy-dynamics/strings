# Domain Playbooks

Use a domain playbook whenever Lovasz recognizes the mathematical area. The point is to start with the right obstruction families, theorem families, examples, and computation types.

## Selection

```markdown
### Domain Selection
- Primary domain:
- Secondary domains:
- Why:
- Playbooks loaded:
- Domain-specific barriers:
- Domain-specific experiments:
```

Load at least one specific playbook for frontier problems. If none exists, record the missing playbook in `lovasz.soc`.

## Available Playbooks

- `domain_extremal_graph_theory.md`
- `domain_multiplicative_number_theory.md`
- `domain_ramsey_theory.md`
- `domain_additive_combinatorics.md`

## Output

Each domain pass should produce:

- theorem family shortlist,
- standard extremal examples,
- likely false stronger variants,
- computation/search templates,
- first three lemmas to try,
- known barriers and how they usually appear.
